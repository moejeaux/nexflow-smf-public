# Community AgentCards

This directory contains AgentCards submitted by the community for services built on NexFlow SMF.

## How to Add Your Card

See [SUBMISSION_GUIDE.md](../SUBMISSION_GUIDE.md) for complete instructions.

**Quick steps:**
1. Copy `../template.json`
2. Fill in your service details
3. Save to this directory as `your-service-name.json`
4. Submit a PR

## Requirements

- Service must support x402 payments through NexFlow
- Valid JSON following AgentCard schema
- Working endpoint URLs
- Clear description

## Example Services

Check out `../nexflow/` for examples of well-structured cards.
